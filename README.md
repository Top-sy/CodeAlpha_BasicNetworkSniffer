# CodeAlpha Basic Network Sniffer

## 📌 Project Overview
A Python-based basic network packet sniffer built using Scapy to capture and analyze live network traffic. This project inspects packets and extracts important information such as source IP, destination IP, protocol type, and packet details for cybersecurity learning and network traffic analysis.

## ✨ Features
- Captures live network packets
- Extracts source and destination IP addresses
- Identifies packet protocols
- Displays packet information in real time
- Helps understand network communication flow

## ⚙️ Tools Used
- Python
- Scapy
