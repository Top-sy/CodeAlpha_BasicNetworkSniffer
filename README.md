# CodeAlpha Basic Network Sniffer

## 📌 Description
A Python-based network packet sniffer built using Scapy to capture and analyze live network traffic. It extracts key details like source IP, destination IP, protocol, and packet data for cybersecurity learning.

## ⚙️ Tools Used
- Python
- Scapy

1. Install dependencies:
   pip install scapy

2. Run the script:
   python sniffer.py
